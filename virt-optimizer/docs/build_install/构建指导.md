# 构建指导

## 环境要求

- OS 内核和软件依赖要求：

    1. 物理机/虚拟机需要启用必要的 eBPF 相关配置（openEuler 22.03 默认已开启）。执行以下命令后，需确保输出包含如下配置项：

        ```bash
        zcat /proc/config.gz | grep DEBUG_INFO
        ```

        ```configure
        CONFIG_DEBUG_INFO=y
        CONFIG_DEBUG_INFO_BTF=y
        ```

       > 若当前 OS 未启用 eBPF 相关配置，则需要修改内核参数并重新编译操作系统内核。

    2. 依赖软件包：

       | 软件包             | 版本信息         |
       |:----------------|:-------------|
       | git             | 2.33.0 及以上版本 |
       | cmake           | 3.22.0 及以上版本 |
       | make            | 4.3 及以上版本    |
       | gcc             | 10.3.1 及以上版本 |
       | g++          | 10.3.1 及以上版本 |
       | rapidjson-devel | 0.8.1 及以上版本  |
       | bpftool         | 12.0.1 及以上版本 |
       | libbpf          | 0.8.1 及以上版本  |
       | libbpf-devel    | 0.8.1 及以上版本  |
       | clang           | 12.0.1 及以上版本 |
       | tar             | 1.34 及以上版本   |
       | rpm-build       | 4.18.2 及以上版本 |


## 获取源码

```bash
git clone https://gitcode.com/openeuler/ubs-virt.git
cd ubs-virt/virt-optimizer
```

## 构建项目

```bash
bash build.sh
```

构建产物位于ubs-virt/virt-optimizer/output/release目录下，产物为virt-optimizer-\<version>-aarch64.rpm
