# 构建指导

## 环境要求

- 编译工具要求：CMake ≥ 3.14.1
- OS内核和软件依赖要求：

    1. 确保内核编译时启用了必要的 ebpf 相关配置（Open Euler22.03默认开启）：

        ```configure
        CONFIG_DEBUG_INFO=y
        CONFIG_DEBUG_INFO_BTF=y
        ```

        > 若使用的OS没有开启ebpf，需要修改参数后重新编译内核。
    2. 物理机依赖软件包：`vmtop ≥ 1.1`
    3. 虚拟机依赖软件包：

        |软件包|版本信息|
        |:-----|:-----|
        |perf|5.10.0及以上版本|
        |docker|5.0.3及以上版本|
        |paramiko|2.11.0及以上版本|
        |ijson|3.4.0及以上版本|

## 获取源码

```bash
git clone https://gitcode.com/openeuler/ubs-virt.git
cd ubs-virt/virt-optimizer
```

## 构建项目

```bash
bash build.sh
```

构建产物位于virt-optimizer目录下，产物为virt-optimizer-\<version>-aarch64.rpm
